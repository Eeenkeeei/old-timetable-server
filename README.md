# Сервер для выпускного проекта
Функционал:
- Регистрация новых пользователей
- Валидация введенных данных
- Шифрование паролей пользователей
- Хранение всех данных в базе Mongo DB 
- Авторизация, предоставление токенов
# Стек технологий:
- Node JS
- Mongo DB
- bcrypt
- jwt
- travis-ci -> heroku
